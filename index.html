This webpage application allows the user to apply to various internships and employment postings posted by Employeers. The users will have the ability to apply for that job through our webpage application with the employeer being able to view who applied for their available position.

With a user-freindly UI design, the user is able monitor and observe post made by employeers who are looking for interns or employees.

Interactions with Employeers

You will be able to apply or talk to the employeer who posted the post your most interested in. Enabling the user the ability to not only just apply to the job, but to also provide or ask for additional information.

Any issues arise the user will be able to contact support which will work tirelessly to resolve the given issue.
